# Changelog

## 1.0.0 — 2026-07-17

- Formalized formulations A, B, C and D and their domains of equivalence.
- Added an exact common validator and 31 regression tests.
- Added direct, centered, common-difference, geometric and elliptic probes.
- Added primitive parametric generation of three-square progressions.
- Validated the parametric and quadratic catalogues object by object at
  `R=100000` on 122,640 progressions.
- Added pilot and confirmatory benchmark artifacts.
- Added the French research manuscript and bibliography notes.
- Updated the problem status through April 2026, distinguishing the
  peer-reviewed result for all orders `n>=4` from a recent unrefereed claim for
  order 3.
- Completed the final terminology, citation and reproducibility audit.
- Reserved Zenodo DOI `10.5281/zenodo.21418008` for the v1.0.0 deposit.

The package is released under the MIT License. Its bounded computational result
does not establish global existence or non-existence.
