# Limites

Cette recherche ne constitue pas une preuve d’impossibilité générale.

Elle explore une famille structurée :

- centre carré ;
- progressions arithmétiques de carrés autour du centre ;
- exclusion des dilatations triviales par facteur carré commun.

Pour le cas 8/9, une autre famille importante reste à explorer :

- centre non carré ;
- huit cases extérieures carrées.

Cette famille fera l’objet d’une prochaine branche de recherche.
